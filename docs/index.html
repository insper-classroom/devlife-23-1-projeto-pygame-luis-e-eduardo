Descrição do Jogo


No jogo, você assume o papel de um Kirb e tem como missão completar todas as fases. Para alcançar este objetivo, você precisa coletar todas as estrelas do mapa e avançar para a próxima fase. Fique atento, pois existem inimigos que vão tentar impedir o seu progresso e você precisa derrotá-los para continuar. Esteja preparado para enfrentar os desafios que surgirem em seu caminho!


Esta é a estrela! Para avançar para a próxima fase, é necessário coletar todas as estrelas que estiverem em sua tela. Certifique-se de pegar todas elas para prosseguir.


Este é o Kirb, o personagem que representa você no jogo.


Durante o jogo, você encontrará alguns macaquinhos que tentarão te impedir de pegar as estrelas e avançar para a próxima fase. Caso necessário, não hesite em eliminá-los para prosseguir em sua jornada.

Instruções para executar o jogo


  1. Para clonar o repositório do jogo no Github, utilize o comando git clone seguido do link do repositório.
  2. Entre na pasta onde os arquivos do jogo estão armazenados usando o comando CD.
  3. Instale o Pygame utilizando o comando pip install pygame.
  4. Para executar o jogo, utilize o comando python funcoes.py.
